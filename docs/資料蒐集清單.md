# 蒐集了哪些資料、如何蒐集

以下依「資料類別 → 儲存表 → 來源與蒐集方式」條列，供簡報「主要功能」或「資料規模／資料來源」使用。簡報可只取三欄精簡呈現。

| 資料類別 | 儲存資料表 | 蒐集方式（來源／方法） |
|----------|------------|------------------------|
| 景氣燈號 | business_cycle_data | **CSV**：手動/定期放置 `business_cycle/景氣指標與燈號.csv`，由 CycleDataCollector 讀取並將月資料展開為交易日寫入。主選單選項 1。 |
| 領先指標構成項目 | leading_indicators_data | **CSV**：`business_cycle/領先指標構成項目.csv`，IndicatorDataCollector.import_all_indicators 匯入（選項 1 一鍵匯入）。 |
| 同時指標構成項目 | coincident_indicators_data | **CSV**：`business_cycle/同時指標構成項目.csv`，同上。 |
| 落後指標構成項目 | lagging_indicators_data | **CSV**：`business_cycle/落後指標構成項目.csv`，同上。 |
| 綜合指標與燈號 | composite_indicators_data | **CSV**：`business_cycle/景氣指標與燈號.csv`（綜合指數與燈號欄位），同上；另由同一模組計算衍伸欄位並寫回同表（選項 1 一鍵 / Orange Pipeline Phase 2）。 |
| 景氣對策信號構成項目 | business_cycle_signal_components_data | **CSV**：`business_cycle/景氣對策信號構成項目.csv`，同上。 |
| 上市股票與 ETF 股價 | tw_stock_price_data | **證交所 API**：`MI_INDEX`，StockDataCollector.batch_fetch_stock_prices_only 依日期批次取得（選項 2 / 5 / Orange Pipeline Phase 1）。 |
| 上櫃股票股價 | tw_otc_stock_price_data | **櫃買中心 API**：dailyQuotes，OTCDataCollector.batch_fetch_otc_data 依日期批次取得（選項 5 / Orange Pipeline Phase 1，可選 --no-otc 略過）。 |
| 價格指數 | tw_price_indices_data | **證交所 API**：同證交所行情介面，StockDataCollector 取得價格指數。 |
| 報酬指數 | tw_return_indices_data | **證交所 API**：同上，取得報酬指數。 |
| 大盤融資融券 | market_margin_data | **證交所 API**：`MI_MARGN`，MarginDataCollector.batch_fetch_margin_data 依日期區間批次取得（選項 13 / Orange Pipeline Phase 1）；衍生欄位由同模組計算後寫回同表（Phase 2）。 |
| VIX 原始（期交所） | TFE_VIX_data | **期交所**：getVixData 下載 txt，main.download_vix_data + parse_and_save_to_tfe_vix 解析寫入（選項 14 / Orange Pipeline Phase 1 當月缺失日）。 |
| VIX 月 K 線與衍生 | VIX_data | **衍生**：由 TFE_VIX_data 計算當月月 K 線寫入 VIX_data；衍生欄位由選項 15 / vix_derivatives 寫回同表。 |
| M1B 年增率等 | leading_indicators_data（新增欄位） | **衍生**：讀取 leading_indicators_data 的 m1b_money_supply，M1BCalculator.calculate_and_update 計算年增率、動能等並更新同表（選項 1 一鍵步驟 4 / Orange Pipeline Phase 2）。 |
| 合併總經指標 | merged_economic_indicators | **衍生**：由景氣相關表（領先/同時/落後/綜合/燈號等）合併計算，IndicatorDataCollector.calculate_and_save_merged_indicators（選項 1 一鍵 / Orange Pipeline Phase 2）。 |
| 綜合指標衍伸 | composite_indicators_data（更新欄位） | **衍生**：由 composite_indicators_data 計算月變動、變化率、前 1/2 期、3/6 月均線等（選項 1 一鍵 / Orange Pipeline Phase 2）。 |
| 技術指標（日線/月線） | stock_technical_indicators、stock_technical_indicators_monthly | **衍生**：由 tw_stock_price_data / tw_otc_stock_price_data 計算，TechnicalIndicatorCalculator（選項 17 / Orange Pipeline Phase 2）。 |
| 006208 月線與未來 1 月欄位 | etf_006208_monthly_future | **衍生／匯出**：由專案匯出或計算流程產出，用於 Orange 月線分析與預測。 |
